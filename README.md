# Twitter Sentiment Analysis

## Overview

This project performs sentiment analysis on Twitter data using Natural Language Processing (NLP) and Machine Learning techniques. The model classifies tweets as Positive or Negative.

## Technologies Used

* Python
* Pandas
* NLTK
* Scikit-learn
* TF-IDF Vectorization
* Logistic Regression

## Dataset

Sentiment140 Dataset containing 1.6 million tweets.

## Methodology

1. Data Cleaning
2. Stopword Removal
3. Stemming
4. TF-IDF Feature Extraction
5. Logistic Regression Training
6. Sentiment Prediction

## Results

* Accuracy: 75.1%
* Successful classification of user-entered tweets.
